Quick Start

Get from zero to a complete ECL calculation in under 5 minutes.

Prerequisites This guide assumes you have already installed the application and it is running. If not, follow the installation guide first.
1

Deploy the App

Deploy the application to your Databricks workspace:

databricks apps deploy ifrs9-ecl --source-code-path ./app

Wait for the app to start (usually 30–60 seconds). Open the app URL shown in the CLI output.

2

Run the Setup Wizard

On first launch, you'll see the Setup Wizard. It walks you through:

  1. Connection Test — verifies Lakebase connectivity
  2. Table Validation — checks that required Unity Catalog tables exist
  3. Sample Data — optionally seeds ~84,000 synthetic loans across 5 product types
  4. Configuration — sets default scenario weights, LGD parameters, and chart of accounts

Click "Seed Sample Data" to load the demonstration dataset, then "Complete Setup".

Or via the API:

curl -X POST https://<app-url>/api/setup/seed-sample-data
curl -X POST https://<app-url>/api/setup/complete \
  -H "Content-Type: application/json" \
  -d '{"user": "admin"}'
3

Load Sample Data

If you chose to seed sample data, the application creates synthetic loan portfolios with:

Product TypeApprox. LoansDescription
Mortgages~25,000Residential mortgage loans
Personal Loans~20,000Unsecured personal loans
Auto Finance~15,000Vehicle financing
Credit Cards~14,000Revolving credit facilities
SME Lending~10,000Small business loans

Each loan includes PD, LGD, EAD, staging (1/2/3), days past due, borrower segment, and vintage cohort data.

4

Create Your First Project

Navigate to the Create Project page and fill in:

  • Project ID: Q4-2025-ECL
  • Project Name: Q4 2025 ECL Calculation
  • Reporting Date: 2025-12-31

Or via the API:

curl -X POST https://<app-url>/api/projects \
  -H "Content-Type: application/json" \
  -d '{
    "project_id": "Q4-2025-ECL",
    "project_name": "Q4 2025 ECL Calculation",
    "project_type": "ifrs9",
    "reporting_date": "2025-12-31"
  }'
5

Run Through the ECL Workflow

The 8-step workflow guides you through the entire ECL process:

StepNameWhat Happens
1Data & Portfolio ReviewReview portfolio summary, stage distribution, data quality checks
2Model ExecutionRun satellite models, review PD/LGD calibration results
3ECL CalculationView ECL by product, stage, and scenario; run Monte Carlo simulations
4Scenario AnalysisSet scenario weights (base/upside/downside), review weighted ECL
5Management OverlaysAdd post-model adjustments with justification
6Stress TestingReview sensitivity analysis, scenario comparisons, vintage performance
7Reporting & DisclosureGenerate IFRS 7 reports, GL journals, stage migration schedules
8Sign-OffReview audit trail, sign off the ECL calculation

At each step, click "Advance" to move to the next step. The workflow tracks who performed each action and when.
