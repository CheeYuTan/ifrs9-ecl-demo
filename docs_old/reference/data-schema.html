Data Schema

Complete schema for all Lakebase database tables used by the application.

Schema & Prefix All tables are created in the expected_credit_loss schema by default. Unity Catalog source tables use the lb_ prefix. Both are configurable via Admin Configuration.

Workflow Tables

ecl_projects

Stores ECL project state, workflow progress, and audit trail.

ColumnTypeDefaultDescription
project_idTEXT—Primary key. Unique project identifier.
project_nameTEXT—Display name for the project.
project_typeTEXT—Project type (e.g., "ifrs9").
descriptionTEXT—Project description.
reporting_dateTEXT—Reporting period end date.
current_stepINT0Current workflow step (0–7).
step_statusTEXT—JSON array of step completion statuses.
audit_logTEXT—JSON array of audit entries (action, user, timestamp, detail).
overlaysTEXT—JSON array of management overlay objects.
scenario_weightsTEXT—JSON object of scenario → weight mappings.
signed_off_byTEXT—Name of person who signed off.
signed_off_atTIMESTAMP—Sign-off timestamp.
created_atTIMESTAMPNOW()Project creation timestamp.
updated_atTIMESTAMPNOW()Last update timestamp.

Model Registry Tables

model_registry

Central registry for all PD, LGD, EAD, and satellite models.

ColumnTypeDefaultDescription
model_idTEXT—Primary key. Auto-generated UUID.
model_nameTEXT—Model display name.
model_typeTEXT—PD, LGD, EAD, Staging, or Satellite.

descriptionTEXT—Model description.
statusTEXT'draft'Lifecycle status: draft, validated, approved, champion, retired.
product_typeTEXT—Target product type.
cohortTEXT—Training cohort identifier.
parametersJSONB—Model hyperparameters.
performance_metricsJSONB—Performance metrics (AUC, Gini, etc.).
training_data_infoJSONB—Training data metadata.
is_championBOOLEANFALSEWhether this is the current champion model.
created_byTEXT'system'Creator identity.
created_atTIMESTAMPNOW()Registration timestamp.
approved_byTEXT—Approver identity.
approved_atTIMESTAMP—Approval timestamp.
promoted_byTEXT—Who promoted to champion.
promoted_atTIMESTAMP—Champion promotion timestamp.

retired_atTIMESTAMP—Retirement timestamp.

model_registry_audit

Audit trail for all model status changes.

ColumnTypeDefaultDescription

performed_byTEXT—User who performed the action.

model_runs

Records of satellite model execution runs.

model_changesJSONB—ECL impact of model parameter changes.
