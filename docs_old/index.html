IFRS 9 ECL Workspace

Enterprise-grade Expected Credit Loss platform on Databricks. Calculate, govern, and report ECL with full auditability — powered by Lakebase and Unity Catalog.

Get Started → View Documentation

Key Capabilities

📊

ECL Calculation

Monte Carlo simulation engine with configurable PD/LGD parameters, scenario weighting, and management overlays across an 8-step auditable workflow.

🏛️

Model Governance

Full model registry with version control, champion/challenger tracking, approval workflows, and audit trails for regulatory compliance.

📒

GL Integration

Automated journal entry generation with double-entry accounting, chart of accounts management, posting, reversal, and trial balance reconciliation.

📋

Regulatory Reporting

IFRS 7 disclosure packages, ECL movement schedules, stage migration matrices, sensitivity analysis, and concentration risk reports.

🔬

Backtesting

Traffic-light framework with AUC, Gini, KS, PSI, and Brier score metrics. Cohort-level analysis and trend monitoring over time.

⚡

Advanced Analytics

Markov chain transition matrices, hazard models (Cox PH, discrete-time, Kaplan-Meier), cure rate analysis, CCF estimation, and collateral haircuts.

Built on the Databricks Lakehouse

Deploys as a Databricks App with Lakebase (managed PostgreSQL) for transactional state, Unity Catalog for data governance, and serverless compute for heavy workloads.

🗄️ Lakebase

Managed PostgreSQL for workflow state, model registry, journals, and configuration — with automatic OAuth token refresh.

🔐 Unity Catalog

Loan portfolio data governed by Unity Catalog. Column-level lineage, access controls, and data quality built in.

⚙️ Databricks Jobs

Satellite model training and ECL calculation pipelines run as managed Databricks Jobs with monitoring and alerting.
