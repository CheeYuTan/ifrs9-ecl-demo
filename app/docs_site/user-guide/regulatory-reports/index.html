Regulatory Reports

Generate, view, finalise, and export IFRS 7 disclosure packages and regulatory reports.

Purpose​

The application produces five types of regulatory reports aligned with IFRS 7 disclosure requirements. Reports can be generated in draft, reviewed, finalised, and exported as CSV.

Report Types​

ReportDescription
IFRS 7 DisclosureFull credit risk disclosure package per IFRS 7.35F-35N
ECL MovementPeriod-over-period ECL movement analysis
Stage MigrationStage transition matrix showing loan migration patterns
Sensitivity AnalysisECL sensitivity to key risk parameter changes
Concentration RiskPortfolio concentration analysis by product, stage, and segment

Key Metrics (KPI Row)​

MetricDescription
Total ReportsAll generated reports
DraftReports awaiting review
FinalFinalised reports
SubmittedReports submitted to regulators

Generating Reports​

  1. Click one of the 5 report type cards in the Generate Reports section.
  2. The report is generated and appears in the table below.
  3. Reports are created in Draft status.

Report Table​

ColumnDescription
Report IDUnique identifier
TypeReport type (colour-coded badge)
Report DateAs-of date
StatusDraft, Final, or Submitted
Generated ByUser who generated the report
Created AtGeneration timestamp

Actions per report:

  • View (eye icon) — Opens the full report viewer
  • Export CSV (download icon) — Downloads the report data
  • Finalize (lock icon) — Locks the report as final (draft reports only)

Report Viewer​

When viewing a report, you see:

  • Report metadata (date, project, generated by, timestamp)
  • Section tabs — Each report contains multiple data sections, shown as tabs with row count badges
  • Section tables — Auto-formatted data tables for each section
  • Export CSV button — Export the current view
  • Finalize Report button — Lock as final (draft only)

Filter the report table by type using the dropdown filter.
